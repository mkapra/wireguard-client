WireGuard Connection Manager

The WireGuard Connection Manager, WGCM for short, is an API-Based software to automate and abstract the creation, management, access control and statistic analysis of a sophisticated WireGuard VPN installation.

What you are looking at is the offical HTML, CSS and Javascript based frontend application to control the underlying API. This desing makes it possible to use any other compliant software to integrate with WGCM.

In its current form the API and also this frontend application do not support all planned featuers and may behave unexpectedly. This version is not ready for production use.
